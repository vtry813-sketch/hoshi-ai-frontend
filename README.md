# HOSHI AI

An intelligent AI assistant powered by Google's Gemini AI, featuring chat functionality and image generation.

## Features

- **User Authentication**: Signup, login with JWT
- **AI Chat**: Powered by Google Gemini AI
- **Image Generation**: Fallback image generation API
- **Conversation Management**: Save and manage chat history
- **Admin Panel**: Secure admin access for inconnu boy
- **Responsive Design**: Clean ChatGPT-like interface

## Setup


### Frontend
1. Navigate to `frontend` folder
2. Install dependencies: `npm install`
3. Start frontend: `npm start`
